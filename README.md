# Iyaad's Personal Portfolio Website

This is my fallout inspired CRT terminal style portfolio website I build using React.js.

## Features

- CRT Terminal with aesthetic scanline overlay.
- Modular reuseable components.
- Clean maintainable structure for codebase.
- Terminal on main page for navigation via commands.
- Responsive Dynamic scaling to all screen sizes on PC.
- Pages for: Home(About me), Education, Experience, Projects, Contact.

## Tech Stack
- React.js
- HTML/ Tailwind CSS
- Javascript

### Prerequisites
- npm 
- Node.js

## Available Scripts

### `npm start`
Runs the app in development mode at [http://localhost:3000](http://localhost:3000). The page reloads automatically when you make changes.

### `npm run build`
Builds the app for production into the `build` folder. Bundles and optimizes React for best performance.

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run eject`
**Note: this is a one way operation and cannot be undone.** Exposes all configuration files for full control over the build setup.
